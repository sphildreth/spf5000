# ADR 0002: Use React + TypeScript + Vite for Frontend

- Status: Accepted
- Date: 2026-03-15

## Context
The project requires a very small but maintainable web interface for administration and display rendering. Coding-agent ergonomics are an important consideration.

## Decision
Use React with TypeScript and Vite.

## Consequences
- Broad ecosystem and training-data coverage for coding agents.
- Rapid local development and straightforward production build.
- Slightly more frontend tooling than server-rendered templates, but better long-term flexibility.
